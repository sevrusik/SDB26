# Experimental FRC A2A fixtures

This directory contains **non-normative** fixtures for exploratory FRC A2A controls.

Scope:

- candidate failure modes for autonomous/multi-agent behavior,
- draft observability fields and guardrail events,
- early policy-routing examples before standard graduation.

These files are intentionally **outside** the normative v1.0.x contract:

- they are not required for conformance,
- they are not part of the core fixture set used to validate stable schema behavior,
- semantics may change across iterations.

- `docs/FRC_A2A_EXTENSION.md`
- `schemas/frc_a2a_envelope_v0_2_0.json`
- `examples/frc/` (non-experimental fixtures)
